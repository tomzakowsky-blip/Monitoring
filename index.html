📋 Instrukcja konfiguracji (jednorazowa, 5 minut)

Krok 1. Utwórz nowy Google Sheets. Nadaj mu nazwę np. "Monitoring Wizyt".

Krok 2. W Google Sheets kliknij Rozszerzenia → Apps Script.

Krok 3. Usuń domyślny kod i wklej poniższy skrypt:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.data, data.projekt, data.nruslugi, data.firma,
    data.ilosc, data.forma, data.miejsce, data.stala,
    data.swiadek, data.wykon, data.karta, data.uwagi,
    new Date().toLocaleString('pl-PL')
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

Krok 4. Kliknij Wdróż → Nowe wdrożenie → typ: Aplikacja internetowa → "Kto ma dostęp": Wszyscy → Wdróż.

Krok 5. Skopiuj URL wdrożenia (zaczyna się od https://script.google.com/macros/s/...) i wklej go w pole URL poniżej.

Krok 6. W pole "ID Sheets" wklej ID swojego arkusza (z adresu URL).
